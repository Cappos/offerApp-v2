Welcome to My Offers App
Easy web Offers Management System

This is a brief description of the available modules.

view_stream Chapters

In this section you can create default chapters. The chapter can consists of one or more modules

view_module Modules

Here you can create the default modules. You can determine their working hours, price per hour and description

text_fields Pages

This is the place where you can create the default content pages for the offers.

subtitles Offers

This is the heart of the application. A place where you create offer. The offer can consists general information, chapters, files and content pages

work Clients

Here you can view and create customers and see all the offers related to them

group Sellers

The place where sellers information is entered and managed

attach_money Price per hour

Add prices for hours worked here
